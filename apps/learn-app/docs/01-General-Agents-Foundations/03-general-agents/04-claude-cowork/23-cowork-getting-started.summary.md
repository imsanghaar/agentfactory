# Lesson 23: Getting Started with Cowork

## Requirements

- **Subscription**: Claude Pro or Max (free tier doesn't include Cowork)
- **Platform**: macOS (Windows coming soon)
- **Claude Desktop**: Latest version from claude.ai/download

## Setup Steps

1. Install Claude Desktop app
2. Sign in with Pro/Max account
3. Switch to Cowork mode
4. Grant folder access (use dedicated workspace folders)

## Interface Components

- **Conversation Panel**: Chat with Claude, enhanced with file context
- **Execution Panel**: Shows what Claude is doing, requires approval
- **Artifacts Panel**: Generated documents and results

## Safety Practices

- Create dedicated workspace folders
- Don't grant access to entire home directory
- Keep sensitive documents outside approved folders
- Always review execution plans before approving
