# Lesson 28: Built-in Skills - Documents, Spreadsheets, Presentations

## Built-in Skills Overview

Pre-installed capabilities for common document formats:

| Skill    | File Types | Capabilities                                           |
| -------- | ---------- | ------------------------------------------------------ |
| **docx** | .docx      | Create, edit, tracked changes, preserve formatting     |
| **xlsx** | .xlsx      | Read, analyze, formulas, charts, preserve calculations |
| **pptx** | .pptx      | Create, edit, themes, speaker notes                    |
| **pdf**  | .pdf       | Extract text, identify structure, OCR for scanned docs |

## docx Skill Examples

**Create document**: "Create meeting-notes.docx with title, attendees, agenda items, action items table"

**Edit with tracked changes**: "Replace 'Q1 2026' with 'Q2 2026' using tracked changes"

## xlsx Skill Examples

**Analyze data**: "Read sales-data.xlsx, calculate totals by region, identify top products, add summary tab with charts"

**Formula awareness**: Claude understands existing formulas, creates new ones, preserves dependencies

## pptx Skill Examples

**Create presentation**: "Create product-update.pptx with title slide, overview, 3 feature spotlights, timeline, conclusion"

**Match existing style**: "Update deck-template.pptx with new data while matching existing design"

## When to Use Built-in vs. Custom Skills

**Use built-in for**:

- Standard document formats
- Format preservation needs
- Reliable, tested functionality

**Use custom for**:

- Domain-specific procedures
- Consistent reasoning patterns
- Expertise that applies across formats

**Example combination**: docx Skill for Word creation + custom Skill for "legal contract review" expertise
