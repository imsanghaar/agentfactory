# Chapter 84: Phone & Browser Integration

Connect voice agents to real channels. This chapter builds `voice-telephony` and `web-audio-capture` skills for SIP/Twilio/Telnyx and browser audio/WebRTC.

---

## Goals

- Integrate telephony (SIP/Twilio/Telnyx) with your voice agents
- Capture/process browser audio with Web Audio API and VAD
- Use WebRTC transports for realtime voice
- Capture telephony/browser patterns in reusable skills

---

## Lesson Progression

- Build telephony and web-audio skills
- Phone integration patterns and call flows
- Browser audio capture, VAD, and WebRTC transport
- Capstone: end-to-end channel integration; finalize the skills

---

## Outcome & Method

You finish with phone and browser channels connected to your voice agents plus reusable channel skills.

---

## Prerequisites

- Chapters 79-83 foundations and framework/API skills
